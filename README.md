# React Chessground CRA

A minimal Create React App using **react-chessground** + **chess.js** (no TypeScript).

## Quick start

```bash
npm install
npm start
```

## Features

- Drag & drop with legal move restrictions
- Flip board
- Reset to start
- Shows FEN and SAN move history
- PGN Parsing and Uploads
- PGN Normalization
- Click to Drag Arrows
