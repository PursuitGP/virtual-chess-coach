# Demo checklist

Use this checklist when recording the portfolio walkthrough:

1. Open the deployed application, choose a Lichess rating pool, and select **Try example game**.
2. Show the PGN metadata, board playback, evaluation bar, and graph.
3. Step through several moves while showing Stockfish, Lichess, and motif evidence.
4. Show the automatic evidence and coaching loading stages.
5. Step through the game with the arrow keys or clickable move history.
6. Show a move-aligned explanation and its evidence references.
7. Optionally change coaching perspective to regenerate the review.
8. Briefly show the architecture diagram in the README.

Target length: 60–90 seconds. Avoid presenting generated coaching as
authoritative; describe it as LLM synthesis grounded by stronger chess-specific
systems.
